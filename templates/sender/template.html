Przygotowanie pliku Excel

Dowiedz się, jak poprawnie sformatować i wypełnić plik Excel, aby zaimportować kontakty do kampanii.

Kroki przygotowania

1. Pobierz szablon
2. Uzupełnij dane podstawowe
3. Dodaj kolumny dodatkowe
4. Zapisz plik jako .xlsx
5. Prześlij w panelu
Uwaga: Kolumny First Name, Last Name, Email i Phone są wymagane.

1. Dane podstawowe

Kolumna Opis Przykład nagłówka
First Name Imię odbiorcy Imię, First Name
Last Name Nazwisko odbiorcy Nazwisko, Last Name
Email Adres e-mail Email, E-mail
Phone Numer telefonu Telefon, Phone

2. Kolumny dodatkowe

Możesz dodać dowolne inne pola, np. Firma, Stanowisko, Notatki. System automatycznie je zmapuje i zapisze w extra_data.

Firma
Stanowisko
Notatki

3. Formatowanie

  • Użyj formatu .xlsx.
  • Unikaj scalonych komórek.
  • Usuń puste wiersze/kolumny na końcu arkusza.
  • Data i liczby w formacie tekstowym lub odpowiednim formacie Excela.

4. Przykładowe dane

First Name Last Name Email Phone Firma
Anna Kowalska anna@example.com +48 600 123 456 Example S.A.
Jan Nowak jan.nowak@firma.pl 600 987 654 Firma XYZ

5. Zapisz i prześlij

Po uzupełnieniu danych zapisz plik i przejdź do panelu tworzenia kampanii. 


# Przykład nazwy pliku:
mojakampania_contacts.xlsx

# Upload:

2. Zatwierdź
